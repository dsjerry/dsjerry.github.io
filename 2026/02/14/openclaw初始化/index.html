创建飞书机器人

参照：https://docs.openclaw.ai/channels/feishu#step-1-create-a-feishu-app

接入模型、飞书

  1. 下载飞书插件
  1. 通过 onboard 命令配置（推荐），这个过程会同时配置插件、模型等

  • 确认之后，选择Quick start

如果已经配置过，会提示选择现有或者重新配置

  1. 选择模型提供商，如果列表中没有则选中Custom Provider

然后依次输入 Api Base URL、Api Key

对于 deepseek，应该使用 openai 兼容的api格式，所以在 https://api.deepseek.com 需要再加上 /v1 路径

  1. 选中要配置的飞书插件，填入飞书的 App ID、App Secret

后续可通过 openclaw config 命令来修改配置
