uni-app x

介绍

包括uts语言、uvue渲染引擎、uni的组件和API、以及扩展机制。

uts

uts是一门类ts的、跨平台的、新语言，文件以.uts结尾

多端原生：

  • web/小程序：JS
  • android：Kotlin
  • ios：Swift
  • 鸿蒙：ArkTS

强约束：

  • 不支持undefined
  • 条件语句必须是boolean
  • 没有any（严格限制）
  • 不能随便写 DOM/BOM

运行机制：

  • ts：最终还是运行js
  • uts：AOT变异性，app端直接原生二进制

≈ 跨端 ts，为了能编译到 Kotlin/Swift/ArkTS 做了强约束。很多 js 特性内容（DOM、webapi、node api等）不能使用，意味着很多 npm ts 包无法使用

可使用必须是 ES 模块、不依赖浏览器 / Node 环境、不使用动态特性。比如lodash-es、dayjs这种纯工具函数等类型

uvue

uvue 代替的是 html、css，文件以.uvue结尾。类比 flutter 生态，可理解为 uts 为 dart，而 uvue 类似 flutter

原生渲染：

  • App 端：不跑 WebView/JS 引擎，直接映射为原生 View（Android→Kotlin View，iOS→Swift UIView）；
  • H5 / 小程序：编译为优化 JS / 小程序代码；

条件编译

在 C 语言中，通过 #ifdef、#ifndef 的方式，为 Windows、Mac 等不同 OS 编译不同的代码。uniapp 中参考了这个做法。

以 #ifdef 或 #ifndef 加 %PLATFORM% 开头，以 #endif 结尾。

  • #ifdef：if defined 仅在某平台存在
  • #ifndef：if not defined 除了某平台均存在
  • %PLATFORM%：平台名称

可以使用与或，比如#ifdef H5 || MP-WEIXIN
