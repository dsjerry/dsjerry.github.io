йҰ–йЎөеҪ’жЎЈеҲҶзұ»ж Үзӯҫе…ідәҺ
2020-06-26еҸ‘иЎЁ2021-01-23жӣҙж–°еҗҺз«ҜеӯҰд№ 16 еҲҶй’ҹиҜ»е®Ң (еӨ§зәҰ2461дёӘеӯ—)0ж¬Ўи®ҝй—®

PHPйқўеҗ‘еҜ№иұЎиҝӣйҳ¶

иҝҷзҜҮеҢ…жӢ¬зұ»еёёйҮҸгҖҒйқҷжҖҒжҲҗе‘ҳгҖҒе°ҒиЈ…гҖҒз»§жүҝгҖҒжҺҘеҸЈ

зұ»еёёйҮҸдёҺйқҷжҖҒжҲҗе‘ҳ

еҰӮжһңеёҢжңӣзұ»дёӯе®ҡд№үзҡ„жҲҗе‘ҳиў«жүҖжңүеҜ№иұЎе…ұдә«пјҢеҸҜд»ҘдҪҝз”Ёзұ»еёёйҮҸжҲ–йқҷжҖҒжҲҗе‘ҳжқҘе®һзҺ°

зұ»еёёйҮҸ

  • е®ҡд№үеҗҺзҡ„зұ»еёёйҮҸеҖјдёҚеҸҳпјҢеңЁзұ»дёӯдҪҝз”Ё const е…ій”®еӯ—е®ҡд№үзұ»еёёйҮҸ

    еңЁи®ҝй—®зұ»еёёйҮҸзҡ„ж—¶еҖҷпјҢдҪҝз”Ёж јејҸ зұ»еҗҚ::еёёйҮҸеҗҚз§° пјҢ**::** з§°дёәиҢғеӣҙи§Јжһҗж“ҚдҪңз¬Ұ

    • еңЁзұ»еҶ…д№ҹеҸҜд»Ҙи®ҝй—®зұ»еёёйҮҸпјҢдҪҝз”Ё self е…ій”®еӯ—д»Јжӣҝзұ»еҗҚ
    • еҰӮ self::SCHOOL пјҢиҝҷж ·йҒҝе…ҚдәҶдҝ®ж”№зұ»еҗҚд№ӢеҗҺиҰҒдҝ®ж”№зұ»дёӯзҡ„д»Јз Ғзҡ„йә»зғҰ

йқҷжҖҒжҲҗе‘ҳ

  • дҪҝз”Ё static е…ій”®еӯ—жқҘдҝ®йҘ°йқҷжҖҒжҲҗе‘ҳпјҢеұһдәҺзұ»зҡ„жҲҗе‘ҳ
  • йҖҡиҝҮзұ»еҗҚзӣҙжҺҘи®ҝй—®пјҢдёҚйңҖиҰҒе®һдҫӢеҢ–еҜ№иұЎ

з»§жүҝе’Ңе°ҒиЈ…

дёәдәҶдҝқжҠӨж•°жҚ®дёҚиў«и°ғз”ЁиҖ…йҡҸж„Ҹдҝ®ж”№гҖҒйҳІжӯўйҮҚеӨҚе®ҡд№ү

з»§жүҝ

  • еңЁзҺ°жңүзҡ„зұ»зҡ„еҹәзЎҖдёҠеҺ»жһ„йҖ дёҖдёӘж–°зҡ„зұ»гҖӮиҝҷдёӘж–°зұ»еҸ«дҪң еӯҗзұ»пјҢеҺҹжңүзҡ„зұ»еҸ«дҪң зҲ¶зұ»
  • дҪҝз”Ё extends е…ій”®еӯ—
  • еҸӘе…Ғи®ёеҚ•з»§жүҝпјҢеҚіжҳҜжҜҸдёӘеӯҗзұ»еҸӘиғҪжңүдёҖдёӘзҲ¶зұ»
  • еҪ“еӯҗзұ»жңүе’ҢзҲ¶зұ»зӣёеҗҢеҗҚеӯ—зҡ„жҲҗе‘ҳзҡ„ж—¶еҖҷпјҢеӯҗзұ»жҲҗе‘ҳдјҡиҰҶзӣ–зҲ¶зұ»жҲҗе‘ҳ
traits е…ій”®еӯ—

  • дҪҝеҫ—PHPеҸҜд»ҘиҮӘз”ұеӨҚз”ЁжҲҗе‘ҳеұһжҖ§е’Ңж–№жі•пјҢеҮҸе°‘еҚ•з»§жүҝзҡ„йҷҗеҲ¶

    еҪ“еӯҗзұ»гҖҒзҲ¶зұ»гҖҒtraitsжңүзӣёеҗҢеҗҚз§°зҡ„жҲҗе‘ҳзҡ„ж—¶еҖҷпјҢдјҳе…Ҳзә§жҳҜпјҡеӯҗзұ» > traits > зҲ¶зұ»

    еҸҜд»Ҙз»ҷ traits дёӯеҗҢеҗҚзҡ„жҲҗе‘ҳеҸ–еҲ«еҗҚпјҢдҫӢеҰӮжңү Cat е’Ң Dog дёӨдёӘTraits дёӯжңүзӣёеҗҢзҡ„ shout()

    • еҪ“жү§иЎҢ cry()пјҢзҡ„ж—¶еҖҷпјҢе®һйҷ…дёҠжҳҜжү§иЎҢ Dogдёӯзҡ„ shout()

е°ҒиЈ…

  • йҡҗи”ҪзЁӢеәҸеҶ…йғЁзҡ„з»ҶиҠӮпјҢд»…еҜ№еӨ–ејҖж”ҫжҺҘеҸЈ
  • зұ»зҡ„е°ҒиЈ…йҖҡиҝҮи®ҝй—®жҺ§еҲ¶дҝ®йҘ°з¬Ұе®һзҺ°зҡ„
    • publicпјҡе…¬жңүдҝ®йҘ°з¬ҰпјҲеҗҢдёҖзұ»еҶ…гҖҒеӯҗзұ»гҖҒзұ»еӨ– еҸҜи®ҝй—®пјү
    • protectedпјҡдҝқжҠӨжҲҗе‘ҳдҝ®йҘ°з¬ҰпјҲеҗҢдёҖзұ»еҶ…гҖҒеӯҗзұ» еҸҜи®ҝй—®пјү
    • privateпјҡз§Ғжңүдҝ®йҘ°з¬ҰпјҲеҗҢдёҖзұ»еҶ… еҸҜи®ҝй—®пјү

еңЁPHP 4 дёӯзҡ„жүҖжңүеұһжҖ§йғҪжҳҜз”Ё var еЈ°жҳҺзҡ„пјҢж•Ҳжһңе’Ң public дёҖж ·пјҢд»ҘеҗҺзҡ„зүҲжң¬д№ҹе…је®№пјҢдҪҶжҳҜдјҡиҪ¬жҚўдёә public

иҰҒжғіи®ҝй—®еҲ° protected е’Ң private зҡ„жҲҗе‘ҳпјҢжңүдёӨз§Қж–№жі•пјҡе…¬жңүж–№жі•пјҢйӯ”жңҜж–№жі•

  • е…¬жңүж–№жі•

    еңЁдёҖдёӘзұ»дёӯеЈ°жҳҺдёҖдёӘе…¬жңүж–№жі•пјҢз„¶еҗҺйҖҡиҝҮиҝҷдёӘе…¬жңүж–№жі•и®ҝй—®гҖӮдҫӢеҰӮпјҡеңЁдёҠйқў User зұ»дёӯеҠ е…Ҙ

    з„¶еҗҺи°ғз”Ё

  • йӯ”жңҜж–№жі•

    еҫ…ж•ҙзҗҶвҖҰвҖҰ

ж–№жі•йҮҚеҶҷ

  • йҮҚеҶҷж–№жі•ж—¶пјҢиҰҒдҝқжҢҒеҸӮж•°ж•°йҮҸдёҖиҮҙ
  • еӯҗзұ»дёӯж–№жі•зҡ„и®ҝй—®зә§еҲ«зӯүдәҺжҲ–иҖ…е°ҸдәҺзҲ¶зұ»дёӯиў«йҮҚеҶҷзҡ„ж–№жі•зҡ„и®ҝй—®зә§еҲ«
йқһйқҷжҖҒж–№жі•йҮҚеҶҷ

  • йӯ”жңҜеёёйҮҸ __CLASS__ з”ЁдәҺиҝ”еӣһеҪ“еүҚиў«и°ғз”Ёзҡ„зұ»еҗҚ

  • з§ҒжңүжҲҗе‘ҳеҸӘиғҪеңЁжң¬зұ»еҶ…и®ҝй—®пјҢжүҖд»ҘзҲ¶зұ»зҡ„з§ҒжңүеұһжҖ§жҲҗе‘ҳдёҚиғҪиў«йҮҚеҶҷ

йқҷжҖҒж–№жі•йҮҚеҶҷ

еҜ№йқҷжҖҒжҲҗе‘ҳзҡ„и°ғз”ЁйҷӨдәҶеҸҜд»ҘдҪҝз”Ёзұ»еҗҚпјҢиҝҳеҸҜд»ҘдҪҝз”ЁselfгҖҒparentгҖҒstatic е…ій”®еӯ—д»Јжӣҝ

  • selfпјҡиҺ·еҸ–еҪ“еүҚж–№жі•и°ғз”Ёж—¶жүҖеңЁзҡ„зұ»

  • parentпјҡиҺ·еҸ–еҪ“еүҚзұ»зҡ„зҲ¶зұ»

  • staticпјҡиҺ·еҸ–е®һйҷ…иҝҗиЎҢж—¶ж–№жі•жүҖеңЁзҡ„зұ»пјҢд№ҹз§°дёәеҗҺжңҹйқҷжҖҒз»‘е®ҡ

final е…ій”®еӯ—

иў« final е…ій”®еӯ—дҝ®йҘ°зҡ„зұ»е’ҢжҲҗе‘ҳж–№жі•дёҚиғҪиў«дҝ®ж”№пјҢfinal зұ»дёҚиғҪиў«з»§жүҝпјҢеҸӘиғҪиў«е®һдҫӢеҢ–гҖӮ

  • еңЁд»Јз ҒдёҠе‘ҠиҜүдәҶеҲ«дәәпјҢеңЁиҝҷйҮҢе·Із»Ҹз»“жқҹдәҶпјҢеңЁд»Јз ҒеұӮйқўдёҠйҷҗеҲ¶дәҶзұ»зҡ„дҪҝз”Ёж–№ејҸпјҢд»ҺиҖҢеҮҸе°‘дёҚеҝ…иҰҒзҡ„жІҹйҖҡ

жҠҪиұЎзұ»е’ҢжҺҘеҸЈ

еңЁйЎ№зӣ®жқҘеҸ‘дёӯпјҢз»ҸеёёйңҖиҰҒе®ҡд№үж–№жі•жқҘжҸҸиҝ°зұ»зҡ„дёҖдәӣиЎҢдёәзү№еҫҒпјҢдҪҶжҳҜиҝҷдәӣиЎҢдёәзү№еҫҒеңЁдёҚеҗҢзҡ„жғ…еҶөд№ӢдёӢеҸҲжңүдёҚеҗҢзҡ„зү№зӮ№гҖӮжүҖд»ҘеңЁиҝҷз§Қж— жі•зЎ®е®ҡзҡ„жғ…еҶөдёӢпјҢе°ұиҰҒдҪҝз”ЁжҠҪиұЎзұ»е’ҢжҺҘеҸЈ

жҠҪиұЎзұ»дёҺжҠҪиұЎж–№жі•

жҠҪиұЎзұ»з”ЁжқҘе®ҡд№үжҹҗз§ҚиЎҢдёәпјҢдҪҶжҳҜе…·дҪ“зҡ„е®һзҺ°йңҖиҰҒ еӯҗзұ» жқҘе®ҢжҲҗгҖӮдҪҝз”Ё abstract е…ій”®еӯ—жқҘдҝ®йҘ°жҠҪиұЎзұ»

жҜ”еҰӮи·‘жӯҘиҝҷдёӘиЎҢдёәпјҢжңүжҒўеӨҚи·‘гҖҒеҹәзЎҖи·‘гҖҒй•ҝи·‘зӯүеӨҡз§Қи·‘жӯҘж–№ејҸ

еңЁдҪҝз”Ёзҡ„ж—¶еҖҷйңҖиҰҒжіЁж„Ҹпјҡ

  • жҠҪиұЎж–№жі•жҳҜеҸӘжңүж–№жі•еЈ°жҳҺиҖҢжІЎжңүж–№жі•дҪ“зҡ„зү№ж®Ҡж–№жі•
  • еҗ«жңүжҠҪиұЎж–№жі•зҡ„зұ»еҝ…йЎ»иў«е®ҡд№үдёәжҠҪиұЎзұ»
  • жҠҪиұЎзұ»дёӯеҸҜд»ҘжңүйқһжҠҪиұЎж–№жі•гҖҒжҲҗе‘ҳеұһжҖ§е’ҢеёёйҮҸ
  • жҠҪиұЎзұ»дёҚиғҪиў«е®һдҫӢеҢ–пјҢеҸӘиғҪиў«з»§жүҝ
  • еӯҗзұ»з»§жүҝжҠҪиұЎзұ»ж—¶еҝ…йЎ»е®һзҺ°жҠҪиұЎж–№жі•пјҢеҗҰеҲҷд№ҹеҝ…йЎ»е®ҡд№үжҲҗжҠҪиұЎж–№жі•жңүдёӢдёҖдёӘз»§жүҝе®һзҺ°

жҠҪиұЎзұ»дёӯзҡ„жҠҪиұЎж–№жі•иў«еЈ°жҳҺдёә protectedпјҢйӮЈд№Ҳеӯҗзұ»дёӯе®һзҺ°зҡ„ж–№жі•еҸҜд»ҘеЈ°жҳҺдёә protected жҲ–иҖ…жҳҜ publicпјҢиҖҢдёҚиғҪжҳҜ privateгҖӮе°ұжҳҜиҜҙиҢғеӣҙеҸҳеӨ§дәҶ

  • еңЁдҪҝз”Ёз»§жүҝжҠҪиұЎзұ»зҡ„еӯҗзұ»дёӯпјҢиҝҳеҸҜд»Ҙж·»еҠ е…¶д»–дёңиҘҝгҖӮгҖӮдҪҶжҳҜеңЁе®һзҺ°зҡ„ж—¶еҖҷпјҢеҸӮж•°еҝ…йЎ»е’Ңе®ҡд№үжҠҪиұЎж–№жі•зҡ„еҸӮж•°дёҖж ·
  • жҠҪиұЎе®һзҺ°дәҶпјҢеҪ“з„¶е°ұдёҚз”ЁжҠҪиұЎе•Ұ

жҺҘеҸЈ

еҰӮжһңжҠҪиұЎзұ»дёӯзҡ„ жүҖжңү ж–№жі•йғҪжҳҜ жҠҪиұЎж–№жі•пјҢйӮЈиҝҷдёӘзұ»е°ұеҸ«еҒҡжҺҘеҸЈпјҢе…ій”®еӯ—interface

еңЁжҺҘеҸЈдёӯпјҢжүҖжңүзҡ„ж–№жі•еҸӘиғҪжҳҜе…¬жңүзҡ„пјҢдёҚиғҪдҪҝз”Ё final е…ій”®еӯ—жқҘдҝ®йҘ°

  • и®°еҫ—пјҒжІЎжңүе…·дҪ“зҡ„еҮҪж•°дҪ“пјҒ
  • еӣ дёәжҺҘеҸЈжүҖжңүж–№жі•йғҪжҳҜжҠҪиұЎзҡ„пјҢжүҖд»ҘзңҒз•ҘжҺү abstract е…ій”®еӯ—
  • жҺҘеҸЈж–№жі•дҪ“жІЎжңүе…·дҪ“е®һзҺ°пјҢжүҖд»ҘйңҖиҰҒйҖҡиҝҮжҹҗдёӘзұ»дҪҝз”Ё implements е…ій”®еӯ—жқҘе®һзҺ°жҺҘеҸЈ

  • MobilePhone зұ»дёӯеҝ…йЎ»е®һзҺ° ComInterface жҺҘеҸЈдёӯе®ҡд№үзҡ„жүҖжңүж–№жі•

  • дёҖдёӘзұ»еҸҜд»Ҙе®һзҺ°еӨҡдёӘжҺҘеҸЈпјҢз”ЁйҖ—еҸ·йҡ”ејҖпјҢдҪҶжҳҜжҺҘеҸЈдёӯзҡ„ж–№жі•дёҚиғҪйҮҚе‘ҪеҗҚ

  • жҺҘеҸЈдёӯеҸҜд»Ҙе®ҡд№үеёёйҮҸпјҢдёҺзұ»еёёйҮҸз”Ёжі•зӣёеҗҢпјҢдҪҶжҳҜдёҚиғҪиў«еӯҗзұ»жҲ–еӯҗжҺҘеҸЈиҰҶзӣ–

  • зұ»д№ҹеҸҜд»ҘеңЁз»§жүҝзҡ„ж—¶еҖҷе®һзҺ°жҺҘеҸЈ

extends е’Ң implements зҡ„еҢәеҲ«

  • extendsпјҡз»§жүҝдёҖдёӘзұ»жқҘеҲӣе»әеӯҗзұ»
  • implementsпјҡдёҖдёӘзұ»йҖҡиҝҮиҝҷдёӘе…ій”®еӯ—еЈ°жҳҺиҮӘе·ұдҪҝз”ЁдёҖдёӘжҲ–еӨҡдёӘжҺҘеҸЈпјҢиҰҒйҖҡиҝҮйҮҚеҶҷжүҚиғҪе®һзҺ°

з»§жүҝд№ӢеҗҺдҪҝз”ЁеӨҡдёӘжҺҘеҸЈпјҡ

жҲ‘ж„ҹи§үе°ұжҳҜпјҢextends дҪҝз”ЁеңЁдёҚжҳҜе…ЁжҳҜжҠҪиұЎзұ»зҡ„ең°ж–№пјҢеҸҜд»Ҙе®һзҺ°жҷ®йҖҡз»§жүҝпјҢд№ҹеҸҜд»Ҙе®һзҺ°жҺҘеҸЈз»§жүҝпјҢдёҚиҝҮе°ұжҳҜиҝҳдёҚеңЁжҺҘеҸЈзҡ„иҢғз•ҙд№ӢеҶ…пјҢиӢҘжһңиҝҷдёӘзұ»йҮҢйқўйғҪжҳҜжҠҪиұЎж–№жі•пјҢйӮЈе°ұжҳҜжҺҘеҸЈдәҶпјҢжүҖд»Ҙе°ұиҰҒдҪҝз”Ё implements

еӨҡжҖҒдёҺзұ»еһӢзәҰжқҹ

  • еӨҡжҖҒпјҡе®һзҺ°еҗҢдёҖж“ҚдҪңдҪңз”ЁдәҺдёҚеҗҢзҡ„еҜ№иұЎпјҢдә§з”ҹдёҚеҗҢзҡ„жү§иЎҢж•Ҳжһң
  • зұ»еһӢзәҰжқҹпјҡеңЁжҲҗзЁӢеәҸе®һзҺ°еӨҡжҖҒж—¶йҷҗеҲ¶дј е…Ҙзҡ„еҸӮж•°еҝ…йЎ»жҳҜжҹҗдёӘзұ»жҲ–жҺҘеҸЈ

иҺ·еҸ–е•Ҷе“Ғд»·ж ј

  • $g иЎЁзӨәз”ЁжҲ·дј е…Ҙзҡ„е…·дҪ“е•Ҷе“ҒеҜ№иұЎ
  • дёәдәҶдҝқиҜҒжҜҸдёҖдёӘдј е…Ҙзҡ„еҜ№иұЎеҝ…йЎ»еҗ«жңү getName() ж–№жі•е’Ң getPrice() ж–№жі•пјҢе°ҶеҸӮж•° $g зҡ„зұ»еһӢжҢҮе®ҡдёә Goods жҺҘеҸЈ
